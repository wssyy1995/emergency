# 🏥 急症室模拟器 - 游戏玩法文档

## 一、游戏概述

**急症室模拟器**是一款医院主题的模拟经营类小游戏。玩家扮演医院管理员，需要合理安排病人候诊、分配医生治疗、准备医疗物品，在时间限制内治愈指定数量的病人来完成关卡目标。

- **游戏类型**：模拟经营 / 时间管理
- **画面方向**：横屏
- **核心目标**：在倒计时结束前治愈足够数量的病人

### 2.1 等候区（左侧 35%）
- **护士台**：位于区域上方，显示护士形象
- **座位**：2排共8个座位，病人会优先入座等待
- **排队区**：座位满后，新病人会在排队区等待

### 2.2 治疗区（中间 35%）
- **病床**：2×2共4个床位，病人躺卧接受治疗
- **医生**：站立在病床旁进行治疗
- **治疗托盘**：位于区域底部，显示医生需要的物品

### 2.3 器材室（右侧 30%）
- **药品柜**：4格抽屉，存放药品（抗生素、止痛药、肾上腺素、注射液）
- **器材柜**：4格抽屉，存放器械（AED、绷带、手术剪、体温计）
- **操作按钮**：托盘右侧有「重置」和「发送」按钮

## 三、核心玩法流程

### 3.1 病人进场流程

```
游戏开始 → 病人从左侧进场 → 入座/排队 → 耐心倒计时 → 安排治疗
```

1. **进场阶段**：
   - 前4个病人固定间隔3秒进场
   - 后续病人随机间隔2-5秒进场
   - 等候区满8人时暂停进场

2. **分配座位**：
   - 优先分配到8个座位
   - 座位满后进入4个排队位置
   - 病人会自动移动到目标位置

3. **耐心值系统**：
   - 每个病人有独立的耐心值（时间条）
   - 耐心耗尽后病人会生气（变红）
   - 仍未治疗则病人会离开（游戏结束）

### 3.2 治疗流程

#### 急救治疗（紧急疾病）
适用于**紧急病情**（骨折、严重过敏、异物卡喉、中风）：

```
1. 点击病人 → 拖拽到空病床
2. 医生自动走到病床旁
3. 根据病情，医生会请求1-3个物品
4. 从器材室点击物品添加到托盘
5. 点击「发送」按钮提交物品
6. 医生治疗（3-5秒）
7. 治疗完成，病人离开，获得荣誉点
```

#### 自动治疗（普通/轻微疾病）
适用于**普通/轻微病情**（发烧、外伤、感冒、烫伤）：

```
1. 点击病人 → 拖拽到空病床
2. 病人自动开始治疗
3. 倒计时显示治疗进度
4. 治疗完成，显示 cured 图标
5. 点击 cured 图标让病人离开，获得荣誉点
```

## 七、操作指南

### 7.1 拖拽操作

| 操作 | 动作 | 效果 |
|------|------|------|
| 按住病人 → 拖动到病床 | 拖拽 | 安排病人治疗 |
| 按住病人 → 拖动到其他位置 | 交换 | 交换两个病人位置 |

### 7.2 点击操作

| 点击目标 | 效果 |
|----------|------|
| 器材室抽屉 | 将物品添加到托盘 |
| 托盘内物品 | 从托盘中移除物品 |
| 🔄 重置按钮 | 清空托盘所有物品 |
| ▶️ 发送按钮 | 向医生提交托盘物品 |
| cured 图标 | 让治疗完成的病人离开 |
| 音量按钮 | 切换静音/有声 |

### 7.3 手势操作

- **长按病人**：可以拖拽移动
- **点击**：选择/交互

## 八、游戏策略建议

### 8.1 优先级策略

1. **优先处理紧急病人**：红色标记的病情恶化最快
2. **合理使用自动治疗**：普通/轻微疾病优先使用自动治疗，节省医生资源
3. **保持床位周转**：及时点击 cured 图标让病人离开，腾出床位

### 8.2 效率提升

1. **批量准备物品**：看到医生需求后，一次性准备所有物品再发送
2. **合理安排座位**：把紧急病人安排在容易点击的位置
3. **关注耐心条**：黄色/红色耐心条的病人要优先处理

### 8.3 常见失误

- ❌ 把紧急病人放到床上不管（需要医生治疗）
- ❌ 托盘满了还继续添加物品
- ❌ 治愈后忘记点击 cured 图标，占用床位
- ❌ 送错物品给医生（会浪费时间）
